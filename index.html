How to use these scrapers:

  1. Click a button above to download the script for your desired area
  2. Save the script to a folder on your computer
  3. Create a folder named "credentials" in the same directory as the script
  4. Place your Google service account JSON file in the "credentials" folder
  5. Open Terminal or Command Prompt
  6. Navigate to the folder containing the script
  7. Run the script with: python3 scraper_XXX.py
  8. The script will output a spreadsheet URL you can open in your browser

This tool collects public data from the St. Louis Metropolitan Police Department.
